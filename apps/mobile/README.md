# Fitness Platform Mobile App

React Native mobile application for iOS and Android.

## Setup

1. Install dependencies:
```bash
npm install
```

2. For iOS:
```bash
cd ios && pod install && cd ..
npm run ios
```

3. For Android:
```bash
npm run android
```

## Features

- Authentication
- Community discovery
- Event browsing and RSVP
- Profile management
- Push notifications
